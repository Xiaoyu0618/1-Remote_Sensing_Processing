# Remote_Sensing_Processing

A toolkit of **Python, MATLAB, and R scripts** for geospatial data processing and environmental analysis.

The repository aims to provide **reusable**, **publication-ready** workflows for remote sensing research.


## Functional Overview

| Language | Focus | 
|---------|------|
| **Python** | Statistical computing and raster analysis | 
| **MATLAB** | Time-series analysis & format conversion |
| **R** | LiDAR-based vertical structure extraction | 


## Usage
### 1. Prepare datasets
- Raster stacks, LiDAR point clouds, NetCDF time-series, shapefiles
- Check CRS, resolution, and folder organization
### 2. Run a Script
### 3. Export geospatial and tabular outputs
- GeoTIFFs for visualization or GIS
- Feature tables (CSV) for modeling


## Applications
- Monitoring vegetation/ecosystem dynamics  
- Large-scale environmental raster computation  
- LiDAR-based forest structure characterization  
- Preprocessing inputs for GIS / ML modeling
